BudgetBuddy - Complete Application Documentation
=============================================

1. Models Documentation
---------------------

a) Transaction Model
   - Fields:
     * user (ForeignKey to User)
     * amount (DecimalField)
     * description (CharField)
     * date (DateField)
     * transaction_type (CharField)
   - Methods:
     * __str__: Returns formatted transaction string
     * save: Handles transaction processing

b) ExpenseCategory Model
   - Fields:
     * name (CharField)
     * user (ForeignKey to User)
     * icon (CharField)
     * color (CharField)
     * description (TextField)
     * is_default (BooleanField)
   - Methods:
     * __str__: Returns category name
     * get_icon: Returns category icon

c) Budget Model
   - Fields:
     * user (ForeignKey to User)
     * category (ForeignKey to ExpenseCategory)
     * amount (DecimalField)
     * start_date (DateField)
     * end_date (DateField)
     * is_active (BooleanField)
   - Properties:
     * spent: Calculates total spent
     * remaining: Calculates remaining amount
     * percentage_used: Calculates usage percentage

d) SavingsGoal Model
   - Fields:
     * user (ForeignKey to User)
     * name (CharField)
     * target_amount (DecimalField)
     * current_amount (DecimalField)
     * monthly_contribution (DecimalField)
     * target_date (DateField)
     * auto_debit_enabled (BooleanField)
     * debit_day (IntegerField)
     * debit_time (TimeField)
   - Properties:
     * remaining_amount: Calculates remaining target
     * percentage_complete: Calculates progress
     * months_remaining: Calculates time to goal
